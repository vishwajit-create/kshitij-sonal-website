# Kshitij Sonal Website  
  
A full-stack academic personal website built with Node.js + Express.  
  
## Features  
- 7 public pages: Home, Research, Articles, News, About, Contact, Post View  
- Full Admin Panel: Dashboard, Publish, Manage, Customize, Settings  
- Dark Gold theme with 4 presets  
- No external database - JSON flat file storage  
- Railway/Render ready

node server.js
